# FactorioInformationExtractor

Mod for development that exports data from factorio.
Adds the command `/ap-get-info-dump` to create the required exports in `script-output`.

See `./build.sh` and/or `./run.sh` for how to use this.
You probably don't want to run `./run.sh` without reading it first.

For the next step, see also `../data/import-ap-dump.py`.
